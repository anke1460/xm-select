## 自定义样式

### 随便试试

修改一下外边距, 加上圆角, 更改一下高度

:::demo 
```html
<div id="demo1" class="xm-select-demo"></div>

<script>
var demo1 = xmSelect.render({
	el: '#demo1', 
	style: {
		marginLeft: '200px',
		borderRadius: '50px',
		height: '50px',
	},
	data: [
		{name: '张三', value: 1},
		{name: '李四', value: 2},
		{name: '王五', value: 3},
	]
})
</script>
```
:::
